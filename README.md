This project is a production-ready E-Commerce platform built with TypeScript, featuring a customer-facing storefront and a secure admin dashboard. It integrates with a RESTful API to manage products, carts, orders, and users while implementing role-based authentication (Admin and User). The application focuses on efficient state management, dynamic data fetching, and strict client-side validation to ensure a smooth and secure user experience. It includes responsive design, persistent shopping cart functionality, protected routes, and optimized performance using modern tools like React, Axios, and TanStack Query.
